I. 페르소나 설계 프레임워크 [v2 신설]

────────────────────────────────────────────────────────────────
1. 아티스트 웨이 기반 심리 분석 프레임워크
────────────────────────────────────────────────────────────────

  ★ v2에서는 각 페르소나를 아래 5개 차원으로 분석

  [1] 내면의 검열자 (Inner Critic)
      — 그 사람 안에서 자기를 비판하는 목소리는 뭐라고 하는가?
      — 이 목소리가 자기탐색을 막고 있는가?

  [2] 자기 허락 (Permission)
      — 어떤 허락을 스스로에게 하지 못하고 있는가?
      — "~해도 괜찮다"는 말이 필요한 영역은?

  [3] 막혀 있는 것 (Creative/Emotional Block)
      — 원래 있던 감정·욕구·방향성 중 억눌려 있는 것은?
      — "회복"해야 할 것은 무엇인가?

  [4] 안전한 공간의 필요 (Safe Container)
      — 판단 없이 자기를 드러낼 수 있는 공간이 있는가?
      — 마음 워크숍이 그 공간이 되려면 무엇이 필요한가?

  [5] 작은 행동 (Small Action)
      — 지금 바로 할 수 있는 가장 작은 자기돌봄은 뭔가?
      — 실천 가이드가 이 사람에게 실제로 가능한가?

  ■ 감정 계층 모델 (표면 → 깊은 니즈)

  ┌──────────────────────────────────────┐
  │  표면 행동    "심리테스트 해볼까"     │  ← 사용자가 보이는 것
  ├──────────────────────────────────────┤
  │  표면 감정    호기심, 심심함           │  ← 사용자가 말하는 것
  ├──────────────────────────────────────┤
  │  중간 니즈    자기 이해, 재미          │  ← 사용자가 원하는 것
  ├──────────────────────────────────────┤
  │  깊은 니즈    "나는 괜찮은 사람인가"   │  ← 사용자가 말 못하는 것
  │               "내 감정도 중요한가"     │
  │               "나도 돌봄받을 자격이    │
  │                있는가"                 │
  └──────────────────────────────────────┘

  ★ 아티스트 웨이의 통찰:
  사람들이 "창의성 워크숍"을 찾는 진짜 이유는
  창의성이 아니라 "자기 자신과의 관계 회복"이었다.
  마찬가지로, 사용자가 "심리테스트"를 찾는 진짜 이유는
  테스트 결과가 아니라 "나를 알아봐 주는 경험"이다.

────────────────────────────────────────────────────────────────
페르소나 1: 김하늘 — "축제에서 우연히 만난 나를 위한 시간"
────────────────────────────────────────────────────────────────

■ 기본 정보
  이름: 김하늘 (가명)
  나이: 23세
  직업: 대학교 3학년 (심리학 부전공)
  거주: 경기도 수원

■ 디지털 환경 (v1 동일)
  주 사용 기기: 스마트폰 (아이폰)
  주 SNS: 인스타그램 스토리, 카카오톡
  콘텐츠 소비: 숏폼 영상, 심리테스트, 카드뉴스

■ 심리·정서 상태
  - 전공 선택 고민, 졸업 후 진로 불안
  - "나는 뭘 좋아하는 걸까"라는 막연한 자기탐색 욕구
  - MBTI 테스트는 수십 번 해봤지만 "그래서 뭐?" 라는 느낌
  - 전문 상담은 아직 필요성을 못 느낌 (비용도 부담)

■ 아티스트 웨이 심리 분석 [v2 신설]

  [내면의 검열자 목소리]
  "심리학 부전공이면서 자기 마음도 잘 모르잖아."
  "다른 애들은 이미 진로를 정했는데 넌 아직도."
  "이런 거 해봤자 뭐가 달라지겠어."

  [허락하지 못하는 것]
  - "아직 모르겠다"고 말해도 괜찮다는 허락
  - 방향이 없어도 불안해하지 않아도 된다는 허락
  - 재미로 시작해도 의미가 있을 수 있다는 허락

  [막혀 있는 것]
  - 어린 시절의 자유로운 호기심 (지금은 "쓸모 있는" 것만 추구)
  - "좋아하는 것"보다 "해야 하는 것" 중심의 삶 패턴
  - 원작 대응: 2주차(정체성) + 5주차(가능성)

  [안전한 공간의 필요]
  - 친구들 앞에서 "나 사실 잘 모르겠어"라고 말하기 어려움
  - 혼자만의 공간에서 판단 없이 자기를 탐색하고 싶음
  - 마음 워크숍: "아무도 안 보는" 데이터 미수집이 안전감 제공

  [가능한 작은 행동]
  - 좋아하는 음악 들으며 10분 산책 (아티스트 데이트 축소판)
  - 어린 시절 좋아했던 것 하나 떠올려보기
  - 노트 앱에 오늘 느낀 감정 한 줄 적기

■ 감정 계층

  표면: "축제에서 재밌는 거 해볼까"
  중간: "나를 좀 더 알고 싶다"
  깊은: "나는 이대로 괜찮은 걸까?"
        "방향 없이 떠돌아도 괜찮은 걸까?"

■ 유입 시나리오 (v1 유지)
  축제 QR → 미니 워크숍 → 인스타 공유 → 재방문 → 챌린지

■ 회복 여정 시나리오 [v2 신설]

  ┌─────────────────────────────────────────────────────────────┐
  │ 1단계: 첫 만남 (축제 현장)                                  │
  │   M02 30초 감정 체크 → "오, 내 감정을 이렇게 보는 거구나"  │
  │   → 결과 카드 "지금 느끼는 감정, 뭐든 괜찮아요"            │
  │   → "뭐든 괜찮다"는 메시지에 작은 안도감                   │
  │                                                             │
  │ 2단계: 호기심 (집에서 재방문)                                │
  │   F01 감정 날씨 체크 → 감정에 이름 붙이는 경험              │
  │   F10 가치관 카드 → "나는 뭘 중요하게 여기지?"              │
  │   → 단순 유형 분류가 아닌 자기 성찰이 일어남                │
  │                                                             │
  │ 3단계: 발견 (챌린지 시작)                                    │
  │   C04 나를 만나는 12일 챌린지                                │
  │   Day 1: 감정 허용 → "내가 불안한 건 자연스러운 거구나"     │
  │   Day 2: 나는 누구인가 → "나는 이런 사람이구나"             │
  │   Day 5: 가능성 → "두려움 없다면 뭘 하고 싶을까?"          │
  │   Day 12: 통합 → "12일 동안 발견한 나"                      │
  │   → 검열자의 "아직도 모르잖아" 목소리가 작아짐              │
  │   → "지금 탐색하는 것 자체가 의미 있다" 자기 허락           │
  └─────────────────────────────────────────────────────────────┘

■ 마음 워크숍이 이 사람에게 줄 수 있는 것 [v2 추가]

  아티스트 웨이 원리            마음 워크숍 경험
  ─────────────────────────────────────────────────────────
  자기 허락 (Permission)        "지금 모르는 것도 괜찮아요"
                                도입 화면의 안전한 초대

  정체성 회복 (Identity)        강점 발견, 가치관 카드 →
                                "나는 이런 사람이구나" 발견

  가능성 회복 (Possibility)     "두려움 없다면 뭘 할까" 질문
                                5년 후 엽서 → 미래상 상상

  안전한 공간 (Safe Container)  데이터 미수집, 광고 없음
                                "당신의 답은 저장되지 않아요"

페르소나 2: 이준호 — "점심시간에 잠깐, 나를 위한 5분"

■ 기본 정보 (v1 동일)
  나이: 28세 / IT 스타트업 주니어 개발자
  거주: 서울 성수동

■ 심리·정서 상태 (v1 동일)
  - 번아웃 초기 증상 (의욕 저하, 주말에도 피곤)
  - "이 회사 계속 다녀야 하나" 고민
  - 혼자 정리하고 싶음, 명상앱은 안 맞음

■ 아티스트 웨이 심리 분석 [v2 신설]

  [내면의 검열자 목소리]
  "아직 2년차인데 벌써 힘들어하면 안 되지."
  "다른 사람들은 다 잘 버티는데."
  "쉬고 싶다고? 사치야, 더 열심히 해."

  [허락하지 못하는 것]
  - 지쳐있다고 인정해도 괜찮다는 허락
  - 잠깐 쉬어도 무능한 게 아니라는 허락
  - 이 일이 내 길이 아닐 수 있다고 생각해도 된다는 허락

  [막혀 있는 것]
  - 자기 돌봄 감각 (항상 "생산적이어야 한다"는 압박)
  - 직업적 정체성 외의 자기 모습
  - 원작 대응: 8주차(강인함/자기돌봄) + 4주차(온전함/경계)

  [안전한 공간의 필요]
  - 회사에서 "힘들다"고 말하면 약한 사람으로 보일까 걱정
  - 혼자만의 공간에서 솔직하게 자기 상태 점검하고 싶음
  - 점심시간 5분 = 유일한 "나를 위한 시간"

  [가능한 작은 행동]
  - 퇴근 후 아무 목적 없이 10분 걷기
  - 좋아하는 음료 한 잔 사서 천천히 마시기
  - "오늘 내가 잘한 것 1가지" 떠올려보기

■ 감정 계층

  표면: "번아웃 자가진단 해볼까"
  중간: "내 상태를 객관적으로 알고 싶다"
  깊은: "쉬어도 괜찮은 걸까?"
        "나를 돌봐도 되는 걸까?"

  [허락하지 못하는 것]

  - 박소연은 그 도구를 청년들에게 "전달하는 사람"
  - 마음 워크숍이 "전달 가능한 형태"(QR)로 만들어져야 하는 이유

■ 유입 시나리오에 추가되는 동기 [v2]

  기존 동기: "무료 참여형 프로그램 필요"
  추가 동기: "청년들이 '나를 만나는 시간'을 가졌으면 좋겠다"
  → 박소연 본인도 워크숍을 해보면서 "나에게도 필요했다" 깨달음
  → 개인적 공감 + 직업적 필요가 결합된 강한 도입 동기

■ 기관 대상 가치 제안 [v2 추가]

  "《아티스트 웨이》에서 영감을 받은 자기탐색 프로그램을
   청년축제에서 QR 하나로 무료 제공합니다."
  → 보고서에 기재할 수 있는 프로그램 설명 프레이밍

VIII. 핵심 설계 원칙 [v2 업데이트]
════════════════════════════════════════════════════════════════


────────────────────────────────────────────────────────────────
페르소나에서 도출한 핵심 설계 원칙
────────────────────────────────────────────────────────────────

  [1] "QR 찍고 3초 안에 시작" (v1 유지)

  [2] "결과가 예뻐야 공유한다" (v1 유지)

  [3] "재미있지만 의미 있어야 한다" (v1 유지)

  [4] "축제 현장에서 작동해야 한다" (v1 유지)

  [5] "혼자 있는 사람에게도 따뜻해야 한다" (v1 유지, v2 강화)
      → 아티스트 웨이의 "안전한 공간" 원리 반영
      → 밤에 혼자 하는 사용자(환경 E)를 위한 톤 강화

  [6] "뭐든 괜찮다고 말해주어야 한다" [v2 신설]
      — 아티스트 웨이 핵심 원리: 자기 허락 (Permission)
      — 모든 워크숍 도입 화면에 "~해도 괜찮아요" 메시지
      — 결과가 부정적이어도 "이 워크숍을 해본 것 자체가 의미"
      — 서술형은 반드시 "(선택)" 표시 → 강제 없는 초대
      — 사용자의 내면 검열자를 자극하지 않는 톤

  [7] "아무도 당신의 답을 보지 않습니다" [v2 신설]
      — 아티스트 웨이 핵심 원리: 안전한 공간 (Safe Container)
      — 캐머런의 "아무도 읽지 않는 모닝 페이지"를 디지털로 구현
      — "데이터 수집 없음, 새로고침 시 사라짐" 명시
      — 이것이 솔직한 자기탐색의 전제 조건
      — 최민수·윤서현 페르소나에서 특히 핵심적

  [8] "5분이라도 충분하다" [v2 신설]
      — 아티스트 웨이 30분을 5분으로 축소한 것의 정당성
      — 윤서현: "아티스트 웨이 12주를 못 끝낸 죄책감"
      — 마음 워크숍의 존재 자체가 "작아도 괜찮다"는 메시지
      — 완벽주의 해체 (7주차 원리)를 구조적으로 구현
      — "5분이면 충분합니다"를 도입 화면에 자연스럽게
